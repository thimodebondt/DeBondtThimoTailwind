Week 5 – Introductie VCS, HTML en CSS

Ik heb van vorig jaar al ervaring met HTML en CSS waardoor ik deze week vooral heb gebruikt om mijn kennis op te frissen. Daarnaast heb ik ook kennis gemaakt met github, ik had dit nog nooit gebruikt. Ik vond het heel interessant om te zien hoe versiebeheer werkt en hoe je samen aan een project kan werken. Ik heb ook geleerd hoe je een repository kan aanmaken, bestanden kan uploaden en wijzigingen kan bijhouden. Dit zal zeker van pas komen in toekomstige projecten waar samenwerking belangrijk is. Dus deze week heb ik vooral gebruikt om mijn kennis van HTML en CSS op te frissen en om te leren hoe versiebeheer werkt met github. Ik denk dat deze vaardigheden me zullen helpen bij toekomstige projecten.
